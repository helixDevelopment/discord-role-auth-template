## Discord role based authorization
This project uses Discord Oauth2 to authenticate users, and then uses their roles to determine what content they can access.

### How to run
1. Clone the repo
2. Create a `.env` file in the root directory

### Libraries used
- Next.js (React)
- TailwindCSS
- Prisma
- PostgreSQL
- Discord.js
- SWR
- iron-session
